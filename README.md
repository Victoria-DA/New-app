# Victoria-Lecture8

Este é um app que calcula a propbalilidade de uma moeda tirar cara ou coroa, baseado em uma quantidade de tentativas definidas pelo usuário.

## Estado Atual do Projeto

O projeto está em produção. Publicado no endereço: https://victoria-lecture8-ocwu.onrender.com
